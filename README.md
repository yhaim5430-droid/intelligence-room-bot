# Intelligence Room Bot

## הפעלה ב-Railway

1. צור חשבון ב-railway.app
2. לחץ "New Project" > "Deploy from GitHub repo"
3. העלה את הקבצים ל-GitHub
4. הוסף משתני סביבה:
   - TG_TOKEN
   - ALPHA_KEY  
   - CHAT_ID

## פקודות הבוט
- /start - תפריט ראשי
- /scan - סריקה מלאה
- /watchlist - מחירים חיים
- /briefing - תדריך בוקר
- /help - עזרה
